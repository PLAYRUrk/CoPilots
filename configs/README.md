# CoPilots — библиотека конфигов самолётов

Готовые `copilots.json` для конкретных самолётов. Скопируйте конфиг нужного борта в папку самолёта (рядом с `.acf`) у **всех участников сессии** — файлы должны быть идентичными.

Если для самолёта конфига здесь нет, плагин работает и без него: авто-обнаружение синхронизирует стандартные датарефы X-Plane, а `smartcopilot.cfg` (если есть в папке самолёта) подхватывается как fallback.

## Как сделать конфиг для своего самолёта

Главное правило: **один орган управления — ровно один канал синхронизации** (датареф *или* команда, никогда оба — иначе тумблеры откатываются и перескакивают позиции).

1. **Позиции переключателей — датарефами** (`"mode": "onchange"` по умолчанию), но только если самолёт принимает запись в них. Хороший референс — секция `[TRIGGERS]` SmartCopilot-конфига этого самолёта: всё, что там есть, записью управляется.
2. **Кнопки и Lua-управляемые тумблеры — командами.** Плагин ретранслирует нажатие И удержание (Begin/End), поэтому удерживаемые переключатели (тесты пожарной сигнализации, spring-переключатели) работают корректно.
3. **Состояние, которое ведёт симуляция мастера** (стояночный тормоз, триммеры) — датарефами с `"mode": "continuous"`: их шлёт только физмастер, каждый кадр.
4. **Никогда не добавляйте**: РУДы/реверс/шаг винта (`*throttle*`, `*thrust*lever*`, `*reverse*`), штурвал (`*yoke*`), педальные тормоза, оси (`*/axis/*`) — всё это возит UDP-поток PhysicsSync, TCP-дубль вызывает дёргание рычагов.
5. **`"autoSync"`**: `true` — добавить к списку авто-обнаруженные стандартные `sim/` датарефы (подходит простым самолётам); `false` — синхронизировать строго список из конфига (обязательно для бортов с кастомной логикой типа Zibo/LevelUp — авто-зеркала `sim/*` дерутся с кастомным состоянием и вызывают дрожание ручек).
6. Секции `[CONTINUED]`/`[SEND_BACK]` SmartCopilot-конфигов **не переносите** как есть: `SEND_BACK` — это оси (UDP), `CONTINUED` — выходы симуляции; из него берите только parking brake/trim как `continuous`.

Структура папки: `configs/<разработчик>-<модель>/copilots.json` + `README.md` (версия самолёта, X-Plane, статус, ограничения).
